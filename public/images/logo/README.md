# Logo uploads

Place your **Market Watch** header logo here as **`logo.png`** (this is what `components/Navbar.tsx` loads).

- Replace `logo.png` when you have an update.
- For AVIF or SVG instead, change the `src` in `Navbar.tsx` to match the filename.

The navbar expects a wide horizontal logo; current display size is **300px** wide on desktop and **192px** on mobile (height scales automatically).
